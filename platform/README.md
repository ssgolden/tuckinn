# Tuckinn Platform

Production workspace for the Tuckinn Proper commerce platform.

## Apps

- `apps/storefront`: customer storefront
- `apps/admin`: business admin portal
- `apps/staff`: kitchen and front-of-house operations
- `apps/api`: NestJS backend
- `apps/mobile`: React Native / Expo app

## Packages

- `packages/ui`: shared design system
- `packages/types`: shared domain types
- `packages/api-client`: shared API client layer
- `packages/config`: shared project configuration

## Infrastructure

- PostgreSQL
- Redis
- S3
- Stripe
- Docker Compose for local setup

## Start Point

This scaffold is the production replacement for the legacy flat-file prototype in the project root.
